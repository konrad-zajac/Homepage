The application is divided in to three different layers of access, the functionality of the application depends on which level you are currently on:
Level no. Functions availible User
Level 1 Browsing the Datebase(without preferences), Reporting Unregistered user
Level 2 Browsing the Datebase(with preferences), Reporting and reviewing Registered user
Level 3 Browsing the Datebase(with preferences), Reporting, reviewing, adding, removing, ADMIN user
