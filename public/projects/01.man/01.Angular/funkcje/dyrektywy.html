Directives enable the control of the HTML site with a script,
which is written in angular code
# Directive Commentary
1 ng-app Directive to initialize angular.
2 ng-model Directive which links data from HTML with the data from angular.
3 ng-bind Directive which links data from angular with the data from HTML.
4 ng-init Initializes the variable.
5 ng-click Responds from a data on click.
6 ng-repeat Displays the repeated data.
7 ng-show Shows an element if requirements are met.
8 ng-hide Hides an element if requirements are met.
9 ng-include Links another file.
10 ng-class Sets a class for a given element
