Fibonacci's sequence

The algorithm to display numbers in fibonacci's sequence can be implemented in many ways, I am showcasing two:

  • the full version
  • the short version (using Computation by rounding)
In the full version there are six main steps, if the algorithm should inform about the elapsed time there are three optional steps
