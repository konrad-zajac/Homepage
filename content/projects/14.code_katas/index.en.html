Binary gap

I developed this cata beyond the basic premises (by adding the information on how many groups of numbers there are, the group binary representation, and the length of a group) which is to take a number convert it to binary an return the greatest length of zeros (starting with one, and ending with one).


Examples:
5 - 101 => 1
9 - 1001 => 2
2472 - 1001 1010 1000 => 3

To Roman

This cata translates numbers from arbic to roman, The Romans wrote numbers using letters : I, V, X, L, C, D, M. (notice these letters have lots of straight lines and are hence easy to hack into stone tablets)


Examples:
1 => I, 3 => III, 4 => IV 12 => XII, 999 => CMXCIX

Don't give me five
In this kata you get the start number and the end number of a region and should return the count of all numbers except numbers with a 5 in it. The start and the end number are both inclusive.

Examples:
1,9 -> 1,2,3,4,6,7,8,9 -> Result 8
4,17 -> 4,6,7,8,9,10,11,12,13,14,16,17 -> Result 12
The start number will always be smaller than the end number. Both numbers can be also negative!
Don't give me five
