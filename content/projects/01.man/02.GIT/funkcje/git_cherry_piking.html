Cherry picking is the name of a function, which selects only the needed commits from another branch. (notice that the sha changes on the selected branch)
# 1st argument 2nd argument Option Comment
1 git cherry-pick [sha of the commit] Selects a specific commit.
2 git cherry-pick --edit [sha of the commit] Allows for a different commit message .
3 git cherry-pick --signoff [sha of the commit] Inserts the author who signed of the cherrypicking to the commit message.
multiple cherry-pick
4 git cherry-pick --no-commit [sha of the first commit] [sha of the second commit] Selects two commits.
5 git commit -m [ text of the joined commit ] Selects two commits.
