L.p Przycisk Komentarz
1 git status Komenda zwraca informacje o stanie repozytorium.
2 git diff Komenda pokazuje rożnice.
3 git diff --staged
4 git add [plik] Dodaje plik do "staging area".
5 git reset [file] Wynosi plik ze "staging area".
6 git commit Dodaje plik do "staging area" oraz otwiera domyślny edytor tekstu, aby dodac wiadomość do commita.
7 git commit -m "wiadomosc" Dodaje plik do "staging area" oraz w jednej komendzie wpisuje wiadomość do commita.
