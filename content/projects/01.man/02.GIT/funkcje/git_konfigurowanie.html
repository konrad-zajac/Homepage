Configuring the git environment can be divided in two sections:
  • local
  • global
All commands begin with "git config" followed by a location specifier
# git config location specifier Przycisk Komentarz
1 git config --local user.name "[name]" Ustawia localną nazwe użytkownika
który będzie dołączony do commita.
2 git config --global user.name "[name]" Ustawia globalną nazwe użytkownika.
3 git config --global user.email "[email adress]" Ustawia email który będzie dołączony do commita.
4 git config --global color.ui auto Koloryzuje informacje zwracane przez konsole.
5 git config --global core.autocrlf input Changes carriage returns / life feeds to just line feeds
6 git config --global core.autocrlf true Changes all line feeds to carriage returns / life feeds
7 git config --global core.autocrlf false Does no conversion.
