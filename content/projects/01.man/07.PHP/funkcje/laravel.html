# Code Commentary
Creating a new laravel project can be acompished using two methods
1 composer create-project laravel/laravel $NAME This works after installing
2 larval new $NAME This solution is cleaner than the first one,
but it requires adding this "$PATH:$HOME/.composer/vendor/bin" to the system path.
Routing in laravel can be implemented in two ways:
1 Route::get('/$add’, 'Controller@add_function’); This use of routing requires defining that function in the specific controller.
2 public function funkcja_ktora_ma_byc_odpalona() { return ‘static_string’; }
